# Private Payroll

A private payroll application built with React, TypeScript, and modern web technologies.

## Project Info

This is a payroll management system for employers and employees, featuring secure wallet integration and Solana blockchain support.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd private-payroll

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Technologies Used

- **Vite** - Build tool
- **TypeScript** - Type-safe JavaScript
- **React** - UI framework
- **shadcn/ui** - Component library
- **Tailwind CSS** - Styling
- **Solana** - Blockchain integration

## Features

- Employer dashboard for managing payroll
- Employee dashboard for viewing earnings
- Secure wallet integration
- Encrypted data handling

## Deployment

Deploy using your preferred hosting platform (Vercel, Netlify, etc.) or containerize with Docker.
